Objective

File upload and download using TCP.
Exercise

Write a client server socket program in TCP for uploading and downloading files between two different hosts. Also calculate the transfer time in both the cases.
Steps/Hints

    Create two directories in the system where mininet is running
    Xterm mininet hosts and open client and server from two different directories
    Keep a file in the server directory and another file in the client directory
    First, client will download the server file to store in its directory
    Second, client will upload its file to the server directory
    Use time.h to check the transfer time in both the case. Display time taken in the client and server side
